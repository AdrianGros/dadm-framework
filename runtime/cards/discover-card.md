# Discover Card

## Goal

Collect facts about the current state of the milestone.

## Allowed

- inventory systems, files, dependencies, and constraints
- identify risks, assumptions, and open questions
- document evidence and current behavior

## Forbidden

- solution design
- choosing architecture
- writing implementation code as the solution

## Required output

- current-state summary
- scope-relevant facts
- risks and assumptions with severity
- explicit next step into Apply

## Escalate when

- a mandatory human decision trigger fires
- facts are insufficient to define a safe solution boundary
