# Bootstrap Card

## Goal

Initialize a project before milestone execution starts.

## Required sequence

1. Project brief
2. Safety boundaries
3. Scope declaration
4. Milestone plan
5. Plan approval
6. Working mode
7. Discover for `M1`

## Required checkpoint

Do not start `M1` before a milestone plan approval artifact exists.

## Use when

- starting a new project
- restarting a project with unclear setup state
- re-baselining after major plan changes
