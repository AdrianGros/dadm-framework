# Apply Card

## Goal

Turn discovered facts into an explicit solution design.

## Allowed

- architecture and interface design
- data model and workflow definition
- acceptance criteria refinement
- design tradeoff analysis

## Forbidden

- implementation
- hidden scope expansion
- bypassing unresolved medium-or-higher findings

## Required output

- solution design aligned to Discover facts
- assumptions and tradeoffs
- acceptance criteria and planned artifacts
- explicit handoff into Deploy

## Escalate when

- design requires cross-milestone scope change
- design requires unapproved dependency changes
- strategic architecture tradeoff has no clear correct option
